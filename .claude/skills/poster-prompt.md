# poster-prompt.md — ポスター・フライヤー生成プロンプト集

> `.claude/skills/stall-info.md` の情報を前提として使う。
> 出力形式：HTML / SVG / React JSX → スクショでSNS・印刷に転用。

---

## 共通ルール（必ず守る）

```
- カラー版 と 白黒印刷版 を必ずセットで出力する
- 白黒版：グレースケール＋太いアウトライン＋高コントラストで視認性確保
- フォント：日本語は system-ui / 'Hiragino Sans' / 'Noto Sans JP' 優先
- サイズ：A4縦（210×297mm）相当 または SNS正方形（1080×1080px）を指定
- 絵文字・イラスト文字で賑やかさを出してよい（🌭🍞🧂等）
- 価格「200円」「トッピング無料」は必ず目立つ位置に入れる
```

## 保存・index更新ルール

```
1. 生成したHTMLは flyers/ フォルダに保存する
   例: flyers/flyer-color-a4.html
2. 保存後、index.html の該当セクションにカードを追加する
   GENERATED_FLYERS_START〜END の間に <a class="card" href="flyers/xxx.html"> を追記
3. カードのタグは <span class="card-tag tag-done">完成</span>
```

## プロンプトテンプレート集

### A. 基本フライヤー（A4縦・HTML）

```
以下の情報でホットドック屋台のフライヤーをHTMLで作ってください。

【必須情報】
- パート名：おいしいホットドック屋さん
- 旭川高専 システム制御情報工学科 4年
- 日時：10月16日
- 場所：4年S科教室
- メニュー：ホットドッグ 200円
- トッピング無料：ケチャップ・マスタード・マヨネーズ（普通 or 多め）
- 支払い：現金のみ

【デザイン方針】
- ポップで目を引くカラーデザイン（赤・黄・白系）
- ホットドッグのイラストをSVGまたは絵文字で表現
- A4縦（210×297mm）相当のHTMLファイル1枚で完結
- カラー版と白黒印刷版の両方を作ること
```

### C. 店頭メニューボード（横長・SVG）

```
店頭スマートボード表示用のメニューをSVGで作ってください。

【内容】
- タイトル：おいしいホットドック屋さん

```

### D. スタッフ向け調理手順カード（A6・HTML）

```
スタッフが調理中に見る手順カードをHTMLで作ってください。

【手順】
1. ソーセージをフライパンで4分焼く
2. パンに挟んで30秒焼く
3. お客さんの指定トッピングをかける
   - ケチャップ 普通20g / 多め30g
   - マスタード 普通10g / 多め15g
   - マヨネーズ 普通20g / 多め30g

【デザイン】
- A6（105×148mm）印刷想定
- 番号大きく、手順が一目でわかるシンプルデザイン
- 白黒印刷でも読めること
```
